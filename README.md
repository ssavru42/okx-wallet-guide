- บล็อกคำขอ `eth_sign` ที่มีความเสี่ยงสูง (เป็น vector โจมตีบ่อย)
- แสดงคำเตือนสำหรับ Permit / Permit2 approvals และการเปลี่ยน ownership
- แจ้งเตือนที่อยู่ที่คล้ายกัน (similar-address scam)
- ซ่อน airdrop token ที่น่าสงสัย
- มีเครื่องมือจัดการ authorization เพื่อเพิกถอนสิทธิ์ที่ไม่ใช้แล้ว
- รองรับ hardware wallet อย่าง Keystone 3, Ledger และ Trezor สำหรับการเซ็นแบบ isolated

**สิ่งที่คุณต้องรับผิดชอบเอง:**

- เก็บ seed phrase และ private key แบบออฟไลน์ อย่าเก็บในคลาวด์หรือส่งในแชต
- ตรวจทุก approval อย่างละเอียดก่อก่อนเซ็น โดยเฉพาะ contract ที่ไม่คุ้นเคย
